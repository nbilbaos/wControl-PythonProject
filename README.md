# wControl Web / API 🌐

**wControl** es la plataforma de backend y gestión diseñada para centralizar la información de iniciativas tecnológicas y de inversión. Actúa como el núcleo de datos para el ecosistema de aplicaciones de wControl.

## 🚀 Funcionalidades
* **API RESTful:** Endpoints optimizados para la comunicación con clientes móviles y web.
* **Persistencia NoSQL:** Gestión de datos flexible y escalable mediante MongoDB.
* **Despliegue Continuo:** Configurado para actualizaciones automáticas en entornos de nube.
* **Seguridad:** Gestión de certificados y seguridad de borde mediante Cloudflare.

## 💻 Stack Tecnológico
* **Lenguaje:** [Python 3.x](https://www.python.org/)
* **Framework Web:** Flask
* **Base de Datos:** MongoDB
* **Infraestructura:** Render PaaS
* **DNS & Seguridad:** Cloudflare

## 🛠️ Configuración Local
1. Crea un entorno virtual: `python -m venv venv`
2. Activa el entorno e instala las dependencias: `pip install -r requirements.txt`
3. Configura las variables de entorno para la conexión a MongoDB.
4. Ejecuta la aplicación: `python app.py`

## 📊 Propósito del Proyecto

Este sistema forma parte de un conjunto de herramientas tecnológicas orientadas a la gestión eficiente y la formulación de proyectos, integrando análisis de datos y soluciones de software modernas.
